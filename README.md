## Overview
This repository hosts the financial models I built while following the curriculum from **The Valuation School**. The goal is to apply theoretical valuation concepts to real-world companies using Excel.

## Project Contents
- A complete valuation model.
  - *Techniques Used:* Discounted Cash Flow (DCF), Relative Valuation, Forecasting.

## Key Learnings
- **Financial Statement Analysis:** Linking Income Statement, Balance Sheet, and Cash Flow.
- **Forecasting:** Projecting future revenue growth and margins.
- **Valuation:** Calculating WACC (Weighted Average Cost of Capital) and Terminal Value.
